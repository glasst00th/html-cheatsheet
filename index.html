HTML Semantic Elements Reference

Tag Name Description/Purpose
<article> Article Contains independent, self-contained content
<aside> Aside Defines content aside from the content it is placed in (i.e., indirectly related to the surrounding content)
<details> Details Defines additional details that the user can view or hide
<figure> Figure Holds illustrations, diagrams, photos, or other self-contained content
<figcaption> Figure Caption Defines a caption for any <figure> element
<footer> Footer Denotes a footer for a section or document, usually to display authorship, copyright, or contact information
<header> Header Specifies a header for a section or document. You may have several <header> elements in one HTML document, but <header> cannot be placed within a <footer>, <address>, or another <header> element
<main> Main Used to mark the main content of a document
<nav> Navigation Holds a set of navigation links
<section> Section Marks a section within a document. For example, a web page could be split into sections for introduction, content, and contact information.
<summary> Summary A visible heading for a <details> element
<time> Time Used for dates and times
